# Superstore Predictive Modeling Using Machine Learning

## Objective

Build a machine learning model to predict whether an order is profitable.

## Dataset

Sample Superstore Dataset

## Algorithm Used

* Decision Tree Classifier

## Steps Performed

* Data Loading
* Feature Selection
* Train-Test Split
* Model Training
* Prediction
* Accuracy Evaluation
* Confusion Matrix Visualization

## Tools Used

* Google Colab
* Pandas
* Scikit-Learn
* Matplotlib

## Outcome

Successfully built and evaluated a predictive model using supervised machine learning.
